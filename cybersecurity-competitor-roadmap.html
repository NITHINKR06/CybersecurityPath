Phase 1: Building the Foundation

3 Months: October 2025 ‚Äì December 2025

Objective: Master the absolute fundamentals of IT that every cybersecurity professional needs. This phase is non-negotiable and crucial for success.

Month 1: Mastering the Command Line & Operating Systems

üêß Linux/Unix Fundamentals:
Topics:

Learn to navigate the filesystem (ls, cd, pwd), manage files/directories (cp, mv, rm, mkdir), manage permissions (chmod), filter text (grep, awk, sed), and manage processes (ps, kill). Learn about SSH and basic Bash scripting.

Resources:
  • Practice: OverTheWire - Bandit (This is an excellent, game-like way to master the Linux command line)
  • Learning: TryHackMe's Linux Fundamentals module
ü™ü Windows Fundamentals:
Topics:

Get comfortable with Command Prompt and PowerShell. Understand the basics of the Windows Registry, user account control, and the NTFS file system.

Resources:
  • Microsoft's official documentation and tutorials on PowerShell

Month 2: Understanding Networks

Topics:

Learn the OSI and TCP/IP models. Understand what IP addresses, subnets, DNS, and MAC addresses are. Study common protocols like HTTP/HTTPS, FTP, SSH, and ICMP.

Resources:
  • Theory: Professor Messer's CompTIA Network+ Training Course (free videos)
  • Practice: Use Wireshark to capture and analyze your own network traffic. Try the "Intro to Networking" room on TryHackMe

Month 3: Learning to Code for Hackers

üêç Primary Language: Python
Topics:

Focus on fundamentals: variables, data types, loops, conditional statements, functions, and file handling. Learn to use key libraries like requests (for web interaction) and pwntools (for binary exploitation).

Resources:
  • Learning: freeCodeCamp's Python for Everybody course
  • Practice: Codewars (solve small programming challenges)

Topics:

You don't need to be an expert. Just understand basic syntax, pointers, and memory management, as this is crucial for reverse engineering and binary exploitation later.

Phase 2: Introduction to Cybersecurity & Tools

3 Months: January 2026 ‚Äì March 2026

Objective: Transition from general IT to core cybersecurity concepts and begin using the essential tools of the trade.

Month 4: Web Exploitation Fundamentals

Topics:

Learn how websites work (client-side vs. server-side). Dive deep into the OWASP Top 10 vulnerabilities, focusing initially on SQL Injection, Cross-Site Scripting (XSS), and Command Injection.

Tools:

Master the basics of Burp Suite Community Edition. Learn how to intercept, inspect, and modify web traffic.

Resources:
  • Hands-on Academy: PortSwigger's Web Security Academy (The best free resource for learning web hacking)

Month 5: Cryptography & Steganography

Topics:
  • Cryptography: Study classical ciphers (Caesar, Vigen√®re), modern hashing algorithms (MD5, SHA), and common encoding schemes (Base64, URL encoding). Understand the difference between symmetric and asymmetric encryption.
  • Steganography: Learn how data can be hidden within images, audio files, and other media.
Tools:

Get very familiar with CyberChef and command-line tools like openssl.

Resources:
  • Practice: CryptoHack (an outstanding platform for learning cryptography through challenges)

Month 6: Introduction to Reverse Engineering & Forensics

üîç Reverse Engineering:
Topics:

Learn the basics of x86 Assembly. Understand what a disassembler and a debugger do.

Tool:

Download Ghidra (free from the NSA) and learn its basic functions: how to load a binary and view the disassembly and decompiled C code.

üïµÔ∏è Digital Forensics:
Topics:

Learn about file signatures (magic numbers), metadata, and how to analyze disk images and memory dumps.

Tools:

Use command-line tools like strings, exiftool, and binwalk.

Resource:
  • Start with beginner CTF challenges categorized as "Reverse Engineering" or "Forensics" on picoCTF

Phase 3: Deep Dive & Specialization

4 Months: April 2026 ‚Äì July 2026

Objective: Build intermediate skills in the core Hackerverse domains by practicing on dedicated platforms.

Months 7-8: Offensive Security Deep Dive

Focus:

Go beyond the basics in web and binary exploitation. Tackle more complex SQLi, learn about Server-Side Request Forgery (SSRF), and try your first simple buffer overflow.

Resources:
  • Continue with PortSwigger Academy
  • Move on to TryHackMe's "Offensive Pentesting" pathway
  • Start the pwn.college modules for binary exploitation

Months 9-10: Forensics & Reversing Deep Dive

Focus:
  • Forensics: Learn to use Volatility for memory analysis and Autopsy for disk forensics.
  • Reverse Engineering: Practice reading decompiled code in Ghidra to understand program logic and find hidden flags or vulnerabilities.
Resources:
  • TryHackMe has excellent rooms dedicated to Volatility, Autopsy, and Ghidra
  • This is the time to start trying the free, year-long Hackerverse CTF challenges

Phase 4: Consolidation & Competition

2 Months & Beyond: August 2026 ‚Äì Onwards

Objective: Put all your skills together, get comfortable with the pressure of time, and become an active competitor.

Month 11: Full-Spectrum Practice

Activity:

Focus entirely on solving CTF-style challenges. Use CTFtime.org to find active competitions. Join a team or participate solo.

Strategy:

When you get stuck, give it a serious try, but then look up a "write-up" (a solution explanation). Learning from others' solutions is one of the fastest ways to improve.

Month 12 & Onwards: Compete in Hackerverse

Activity:

You are now ready. Actively participate in the monthly Hackerverse CTFs. Your goal is no longer just to learn, but to solve challenges efficiently.

Analyze & Improve:

After each competition, analyze your performance. Which category was your weakest? Spend the next month strengthening that area.

Community:

Join the EC-Council's official Discord server for Hackerverse. Engage with the community, ask questions, and share knowledge.
